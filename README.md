## Краткое описание работы

В ходе лабораторной работы был создан простой TODO-лист (список задач) с использованием HTML, CSS и JavaScript. Изучены основы работы с DOM, обработка событий, динамическое создание элементов и управление состоянием интерфейса.

## Структура проекта

- `index.html` – начальная страница (не используется в основном задании)
- `style.css` – стили для начальной страницы
- `main.js` – скрипт для начальной страницы
- `todo.html` – основная страница TODO-листа
- `todo.css` – стили для TODO-листа
- `todo.js` – логика TODO-листа
- `timer.html` – страница секундомера (дополнительное задание)
- `timer.css` – стили для секундомера
- `timer.js` – логика секундомера
- `img/` – папка со скриншотами
  - `gitPushLab21_DOM_Events_Фамилия.png`
  - `todoBasicLab21_Фамилия.png`
  - `timer_Фамилия.png`
- `README.md` – описание проекта

## Функциональность TODO-листа

- Добавление новых задач через поле ввода и кнопку "Добавить" или нажатие Enter.
- Отметка задачи как выполненной (переключение класса completed).
- Удаление задачи.
- Счетчик общего количества задач.
- Примеры задач при загрузке страницы.

## Дополнительное задание: секундомер

Создано приложение секундомер с функциями:
- Старт (запуск отсчета)
- Стоп (остановка)
- Сброс (обнуление времени)
- Отображение времени в формате ЧЧ:ММ:СС
- Возможность многократного запуска и остановки.
